# Phase Cluster 000: Bootstrap

## Objective

Create the clean source-control spine for CICADA FORGE UE.

## Build type

Mainline bootstrap.

## Scope

Included:
- docs
- project status files
- GitHub hygiene
- Unreal ignore rules
- sidecar folder placeholders
- issue templates
- phase map

Not included:
- Unreal `.uproject` generated files
- plugin source yet
- CAD sidecar implementation
- machine bridge
- live camera bridge
- agent system

## Pass gate

Phase 000 passes when:
- dedicated repo exists
- scaffold committed
- README/status/handoff/masterplan present
- `.gitignore` and `.gitattributes` present
- future ChatGPT session can understand project state by reading repo

## Evidence required

- GitHub repo URL
- first commit hash
- screenshot of local repo in VS Code or GitHub Desktop
